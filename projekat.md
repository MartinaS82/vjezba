# Aplikacija za demonstraciju 

Ovo je AI generisana mini aplikacija za testiranje gita i demonstraciju logike fastapi-ja

Zasada ima samo frontend i backend bez baze
Radimo samo CRUD sa items, trenutno je samo CRD lol jer nema update-a ali pogledajte strukturu! Ovo je buk MVP, pokrecete tako sto u jednom poweshellu pisete:
cd backend
.venv\Scripts\activate
uvicorn main:app --reload

a u drugom:
cd frontend
npm run dev

I onda otiđite na http://localhost:5173 pa cete vidjeti CRUD pocetni

(Napomena, prvo uvijek aktivirajte backend, pa onda frontend.)
